# Cafe Sales Cleaning
## 🧹 Data Cleaning Project with Python and R 📊
<p>I'm pleased to share a project focused on data cleaning using Python and R, based on a synthetic dataset of 10,000 sales transactions from a café. This dataset was intentionally messy, with missing values, inconsistent entries, and formatting issues — perfect for practicing real-world cleaning techniques.</p>

### ✅ Tasks performed:
<ul>
  <li>Renaming columns for clarity</li>
  <li>Fixing data types for numerical fields</li>
  <li>Filling missing values using calculated values</li> 
  <li>Among other cleaning steps</li> 
</ul>

<p>This project helped me strengthen my skills with Pandas and dplyr and reinforced the importance of clean data as the foundation for any solid analysis.</p>

<p>Thanks for reading!</p>

## 🧹 Proyecto de Limpieza de Datos con Python y R 📊
<p>Me complace compartir un proyecto enfocado en la limpieza de datos utilizando Python y R, basado en un conjunto de datos sintético con 10,000 transacciones de ventas en una cafetería. Este dataset fue diseñado intencionalmente con desorden, incluyendo valores faltantes, entradas inconsistentes y problemas de formato, lo que lo hace perfecto para practicar técnicas de limpieza en escenarios del mundo real.</p>

### ✅ Tareas realizadas:
<ul>
  <li>Renombrar columnas para mayor claridad</li>
  <li>Corregir tipos de datos en campos numéricos</li>
  <li>Rellenar valores faltantes usando cálculos</li> 
  <li>Entre otros pasos de limpieza</li> 
</ul>

<p>Este proyecto me ayudó a fortalecer mis habilidades con <code>Pandas y dplyr</code> y reafirmó la importancia de contar con datos limpios como base para cualquier análisis sólido.</p>
